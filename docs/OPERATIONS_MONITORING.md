# Operations Monitoring Center Specification

The **Operations Monitoring Center** aggregates platform health, API latencies, and resource usages.

## Aggregated Diagnostics

1. **Provider Performance Scorecard**: Displays average latency and success rate percentages.
2. **Usage & Cost Tracking**: Displays cumulative tokens used and USD cost metrics.
3. **Background Job Manager Status**: Monitors queued, running, and completed thread logs.
4. **System Diagnostics**: Displays real-time estimates of CPU and Memory loads on the host server.
5. **Safe Mode Recovery**: Displays active status banner if databases are locked.
